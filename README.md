# Jit Malla Portfolio

A cinematic AI research-lab portfolio built with React, Vite, Tailwind CSS, Framer Motion, GSAP, Lenis, Three.js, and React Three Fiber.

## Install

```bash
npm install
```

## Run

```bash
npm run dev
```

## Build

```bash
npm run build
```

## EmailJS

The contact form is integration-ready. Add these environment variables when you have an EmailJS account:

```bash
VITE_EMAILJS_SERVICE_ID=your_service_id
VITE_EMAILJS_TEMPLATE_ID=your_template_id
VITE_EMAILJS_PUBLIC_KEY=your_public_key
```

Without those values, the form uses a polished local success state for development.
